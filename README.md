# Backend — Каталог фильмов

Серверная часть учебного веб-приложения «Каталог фильмов», реализованная на Python с использованием фреймворка Flask.

Backend отвечает за:
- инициализацию приложения;
- работу с базой данных;
- описание моделей данных;
- обработку HTTP-маршрутов;
- наполнение базы тестовыми данными.

## Стек технологий
- Python 3
- Flask
- SQLite (локально)
- Jinja2 (шаблоны используются фронтендом)

## Структура backend
backend/
├── app/
│ ├── init.py # инициализация Flask-приложения
│ ├── models.py # модели данных
│ ├── routes.py # маршруты и обработчики
│ ├── templates/ # HTML-шаблоны
│ └── static/ # статические файлы
├── run.py # точка входа в приложение
├── seed.py # заполнение базы тестовыми данными
└── requirements.txt # зависимости проекта

## Запуск проекта
1. Установить зависимости:
   pip install -r backend/requirements.txt
   
(Опционально) Заполнить базу тестовыми данными:
python backend/seed.py

Запустить сервер:
python backend/run.py
