# Frontend — Каталог фильмов

Пользовательский интерфейс учебного веб-приложения «Каталог фильмов».

Frontend реализован с использованием HTML-шаблонов и статических ресурсов
и отвечает за отображение данных, получаемых от backend-части приложения.

## Назначение
Frontend обеспечивает:
- отображение списка фильмов;
- навигацию по страницам приложения;
- базовую визуализацию данных;
- взаимодействие пользователя с серверной частью через HTML-формы.

## Используемые технологии
- HTML5
- CSS3
- Jinja2 (шаблонизатор Flask)

## Расположение в проекте
Frontend размещён внутри серверного проекта и включает следующие каталоги:

```text
backend/app/
├── templates/   # HTML-шаблоны страниц
└── static/      # CSS, изображения и другие статические ресурсы
```
