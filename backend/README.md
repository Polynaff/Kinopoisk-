# Backend — Каталог фильмов (Flask)

Backend-часть учебного проекта «Каталог фильмов».  
Отвечает за бизнес-логику приложения, маршруты, работу с базой данных и запуск сервера.

## Технологии
- Python 3
- Flask
- Flask-SQLAlchemy
- SQLite

## Структура backend

run.py  
Точка входа в приложение. Запускает Flask-сервер.

seed.py  
Скрипт инициализации базы данных.  
Заполняет базу начальными данными (фильмы и сериалы).

app/__init__.py  
Фабрика приложения (Application Factory).  
Создаёт Flask-приложение, настраивает базу данных, регистрирует маршруты.

app/models.py  
Модели базы данных (SQLAlchemy).  
Содержит модель `Movie`.


app/routes.py  
HTTP-маршруты приложения (Blueprint).  
Реализует:
- главную страницу и поиск,
- страницу фильма,
- страницу избранного,
- добавление и удаление из избранного.

movies.db  
SQLite база данных (создаётся автоматически при первом запуске).

## Запуск backend

1. Установить зависимости:
pip install flask flask-sqlalchemy


2. Инициализировать базу данных:
python seed.py


3. Запустить сервер:
python run.py


Приложение будет доступно по адресу:  
```
http://127.0.0.1:5000
```
